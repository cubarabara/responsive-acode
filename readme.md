## **Responsive**
Description: This tool makes it easier for you to build a Responsive website 

## **How to use it?**
```
1. Hold in editor text "Debug".
2. Run button on Navbar.
3. Crtl-T.
4. open command pallete Search "Responsive".
```

## Feature
* zoom in and zoom out
* support node.js
